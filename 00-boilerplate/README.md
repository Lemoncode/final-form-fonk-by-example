# 00 Boilerplate

In this example we setup the basic plumbing to get up and running our project and launch it in a dev server.

## Steps to build it

- Install project dependencies

```bash
npm install
```

- Start project:

```bash
npm start
```

- Navigate to [http://localhost:3000/](http://localhost:3000/).
